### Burger Tracker
Keep track of the burgers that you want to eat and burgers that you have eaten. Use the "Create Burger!" button to create a burger you haven't tried, mark it off to save it as eaten.

---
### Project Goals
* Create a burger logger with MySQL
* Practice using MVC design pattern.
* Create/delete items from database and display them to DOM.
* Host a MySQL project on Heroku.
